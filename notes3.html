Block and inline elements.

In html , we have two ways of breaking the text into multiple lines.

  1. the <br> empty element:
  2. the <p> non-empty element:

So why do we have both of this? The Difference:

<br> is just ending line! Html doesn't care how many line breaks we put in our page!

But what <p> is doing is , it's creating a box around the content , it tells the browser to create a little invisible box around the text content. This box is called a block!

Blocks are heavily related to how the browser lays out text on the screen!

This block has :

  • Height
  • Width
  • Margin, Above and below it!

This Paragraph element is what is called the Block Element , cuz it create this invisible box called Block around the text content!

Elements that don't create Block are known as inline elements!
