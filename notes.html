WebDev


The layout of a webpage is put together by the browser according to the code in the text file. Text files: Are files that can be edited and could be made of one or more of this components:

  • symbols
  • letters
  • numbers
  • spaces

HTML: Stands for hyper text markup language. HTML document's are building block of a web. It's made up of two main components

  1. Text:That the user will see and read!
  2. Markup: Is a computer language that uses tags to define elements in a document.

In Html, every element start and ends with a tag. The starting tag, the content and the closing tag all together makes up an html element.

Tags

  1. The <mark> tag: To highlight a text with a color

  2. The <sup> tag: Is used to make a superscript on a certain content.

  3. The <sub> tag: used to apply a subscript to a certain content on the web page.

Nested elements

Nesting means: Bits of code that go inside other bits of code! Elements in html go inside other elements, But they have to follow certain rules!

  • <mark> : this element can a inside a <p>. Smaller elements like <mark> Can go inside larger elements like paragraphs and divisions!
  • Paragraph inside paragraph is not possible!
